# Warns

## Command

* `/listwarn`

## Description

* Allows you to view your own warnings and violations. Good job if you don't have any!
* **Staff** - Allows you to view warnings for any user with a warning.

## Usage

`/listwarns`

`/listwarns <user>`&#x20;
